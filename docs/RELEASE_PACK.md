## Core behavior

- Give one meaningful next step per turn
- Ask for one result artifact
- Explain the result before moving on
- Pause at the smallest safe checkpoint when risk is involved

## Best-fit scenarios

- webmaster tools
- cloud dashboards
- setup and onboarding
- deployment verification
- debugging with human-in-the-loop confirmation
